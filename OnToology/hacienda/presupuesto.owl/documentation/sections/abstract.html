Abstract

Este vocabulario se utiliza para la descripción de datos relacionados con los presupuestos de las entidades locales en España, incluyendo gastos e ingresos. Reutiliza términos del vocabulario sobre unidades territoriales que forma parte de este conjunto de ontologías para datos abiertos, y hace referencia a varios esquemas de conceptos SKOS que se basan en las categorías de gastos e ingresos que se definen en la ORDEN EHA/3565/2008, de 3 de diciembre, sobre la estructura de los presupuestos de las entidades locales, y que también se encuentra descrito en la Web http://www.rendiciondecuentas.es/.
